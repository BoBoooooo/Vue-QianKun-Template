# 基座和子应用技术栈

- 基座 main vue2.x
- 子应用 sub-vue vue2.x
- 子应用 sub-react react
- 子应用 sub-vue3 vue3.x
# 开始使用
根目录集成了`npm-run-all`,支持一键执行子项目的script命令。

``` javascript
npm run install
npm start
```

访问 http://localhost:9000 即可

默认 用户名/密码 admin/123

# 部署

``` javascript
npm run build
```
